# 🔥 Lightex Fire Safety - Homepage Prototype

A modern and responsive corporate fire safety website prototype built using HTML, CSS, and JavaScript.

## 🚀 Project Overview

This project is a frontend prototype created for a fire safety company website. It showcases a professional corporate design with multiple sections commonly used in industrial and B2B websites.

## ✨ Features

- Responsive Design
- Sticky Navigation Bar
- Hero Section
- Company Statistics
- About Section
- Product Categories
- Services Section
- Industries We Serve
- Why Choose Us
- Certifications
- Client Showcase
- Testimonials
- Call To Action Banner
- Professional Footer
- WhatsApp Floating Button

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
